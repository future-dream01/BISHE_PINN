   - 模型考虑使用三维的，对三维流场进行反演
   - 训练时采用软约束，推理时采用硬约束，通过对所有边界的软约束来使得控制方程驱动网络达到唯一解，其中壁面采用无滑移条件、对称面采用对称边界条件，入口是压力远场，出口是压力出口
   - 训练策略是分两次，第一次是先纯数据，使得流场快速收敛到合理解，然后再加上RANS控制方程的约束
